# 🚗 Каталог автомобилей (CRUD с консольным вводом)

Консольное приложение для управления каталогом машин в дилерском центре.

## 📋 Описание

Приложение реализует CRUD операции (**С**оздание, **Ч**тение, **О**бновление, **У**даление) для управления каталогом автомобилей. Использует библиотеку readline для интерактивного взаимодействия с пользователем.

## 🔧 Технологии

- **TypeScript** - язык программирования
- **Node.js** - среда выполнения

## 📋 Требования

- Node.js (версия 14 или выше)
- npm (пакетный менеджер)
